# IDE Port Skeptical Review

Status: planning + comparison package.

This folder exists to force a strict, skeptical comparison between:

- `/mnt/DATA/development/projects/repos/fuwa`
- `/mnt/DATA/development/projects/repos/IDE`

The goal is not to celebrate progress. The goal is to identify, without bias:

- what is actually implemented in `fuwa`
- what only looks implemented because of proof scaffolding
- what is still missing relative to the real desktop IDE
- what must be fixed before payload migration can be treated as serious

Contents:

- [comparison.md](comparison.md): feature-by-feature skeptical comparison
- [gaps.md](gaps.md): what is still not ported and why it matters
- [claude-fable5-prompt.md](claude-fable5-prompt.md): one-shot prompt for a
  strong coding model to analyze both repos and stop overclaiming
